# Магазин электрозапчастей (учебный вариант)

Учебная версия проекта для студентов. В ней **намеренно отсутствует модуль заказов** и **частично нарушена ролевая авторизация** для последующей доработки.

## Что уже есть

- каталог электрозапчастей;
- базовые модели каталога;
- интерфейс входа/выхода;
- CRUD товаров для администратора.

## Что намеренно не реализовано

- модуль `orders` полностью удален;
- роли `manager` и `client` отключены (сейчас корректно определяется только `admin`, остальные считаются `guest`).

## Быстрый запуск

1. Активируйте виртуальное окружение.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Выполните миграции:
   ```bash
   python manage.py migrate
   ```
4. Запустите проект:
   ```bash
   python manage.py runserver
   ```

## Основная задача для студента

1. Восстановить модуль заказов (список + CRUD по ролям).
2. Исправить ролевую авторизацию (`guest` / `client` / `manager` / `admin`).
3. Добавить корректные проверки прав и маршрутов.
